## 5. Negative Keywords (Exclusions)
**Exclude these to save budget:**
*   無料 (Unless promoting the free trial aggressively, but might attract non-payers)
*   アプリ (Looking for automated tools, not lessons)
*   求人 (Looking for jobs, not lessons)
*   声優 オーディション (Might be too strictly entertainment focused vs general speaking skills, unless targeting aspiring VAs specifically)

## Recommended Match Types Strategy
*   Start with **Phrase Match** (e.g., "話し方 教室") for core keywords to balance reach and relevance.
*   Use **Exact Match** (e.g., [話し方 教室 広島]) for high-intent location terms.
*   Monitor search terms and add **Negative Keywords** regularly.
